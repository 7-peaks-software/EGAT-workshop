# Lab 01 - Microsoft Copilot Studio Workshops

This lab contains hands-on workshops for learning Microsoft Copilot Studio and building intelligent agents.

## Modules

### 01 - Agent in Copilot Chat
Learn how to build and test agents in Microsoft 365 Copilot Chat.

[Start Module 01](./01-agent-copilot-chat/1-introduction.md)

### 02 - Build a Conversational Agent in Copilot Studio
Learn how to create a conversational agent and ground its answers with knowledge sources in Microsoft Copilot Studio.

[Start Module 02](./02-build-conversational-agent/1-introduction.md)

## Prerequisites

- Access to Microsoft 365 Copilot
- Microsoft Copilot Studio trial license (can be obtained from [Copilot Studio trial sign-up](https://go.microsoft.com/fwlink/?LinkId=2107702))
- Basic understanding of AI and chatbot concepts
