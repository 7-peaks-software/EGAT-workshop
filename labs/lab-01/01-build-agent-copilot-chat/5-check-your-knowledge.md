# Check your knowledge

## Answer the following questions to see what you've learned.

1. **What is the purpose of using a template in Copilot Chat Agent Builder?**

    - To quickly create agents preconfigured for common scenarios.
    - To change the visual theme of the Copilot Chat interface.
    - To manage file uploads and access controls.

2. **In the Copilot Chat agent builder, what is the role of the Instructions section?**

    - It defines the agent's behavior and response style when interacting with users.
    - It provides an onboarding message for new users.
    - It lists the topics available to the agent during chat.

3. **What is one benefit of using starter prompts in an agent?**

    - They help users understand how to interact with the agent by offering example questions.
    - They define the data sources the agent can use during chat.
    - They limit what users are allowed to ask the agent.
