   - Analytics
   - Topics
   - Settings

   - By opening the Test agent pane and asking the triggering questions or similar
   - You don't need to test your agent
   - By reading the topic dialog

   - Copilot
   - Adding XML
   - Creating a new entity with a topic entity linked

   - Both have rich text options to display messages or questions to the user
   - Both are green
   - Both don't have message variations enabled
