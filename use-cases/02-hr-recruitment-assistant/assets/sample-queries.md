# Sample Queries for HR Recruitment Assistant

## Purpose

Use these sample queries to test the HR Recruitment Assistant. Queries cover various recruiter tasks and scenarios.

---

## Category 12: Edge Cases & Error Handling

### Problematic Queries
```
1. [Upload non-CV file] "Analyze this"
2. "Match this CV" (without specifying position)
3. "Find candidates" (no criteria)
4. "Compare candidates" (upload only 1 CV)
5. [Empty query]
```

**Expected Agent Behavior**:
- Request clarification
- Ask for missing information
- Provide helpful error messages
- Suggest correct format

---

## Testing Protocol

### For each query:
1. Enter the query exactly as shown
2. Observe response
3. Verify expected behavior

**Checklist**:
- [ ] Response time < 30 seconds
- [ ] Response is relevant and accurate
- [ ] Information extracted correctly
- [ ] Recommendations are sensible
- [ ] Language (Thai/English) matches query
- [ ] Formatting is clean and professional

---

## Quick Test Script

For rapid testing (10 minutes):

```
1. [English] Upload CV → "Extract information from this CV"
2. [Thai] "จับคู่ CV นี้กับตำแหน่ง ENG-2025-01"
3. Upload 3 CVs → "Compare these candidates"
4. "Find candidates who know Python"
5. "What are the requirements for Senior Engineer position?"
```

All 5 should complete successfully to pass quick test.

## Advanced Test Scenarios

### Scenario A: High Volume Screening
```
Upload 20 CVs for one position
Ask: "Screen all 20 candidates and give me top 5"
Expected: Ranked list with justification
```

### Scenario B: Cross-Position Comparison
```
Upload 1 CV
Ask: "Which of our 5 open positions is this candidate best suited for?"
Expected: Position recommendations with match scores
```

### Scenario C: Missing Information
```
Upload incomplete CV (missing education section)
Ask: "Evaluate this candidate"
Expected: Note gaps, don't hallucinate missing data
```
