# Check your knowledge

Answer the following questions to see what you learned.

1. How does Power Automate connect with Microsoft Copilot Studio?

 - By providing the automation of business processes
 - By acting as a display for customer's data in a table format
 - Via 'Call an Action' selection within the Authoring Canvas in Microsoft Copilot Studio, users can pass variables from user's responses using Power Automate and retrieve data from several different data sources, back to Microsoft Copilot Studio

2. What happens when you select 'Create a Flow' after clicking 'Call an Action' in Microsoft Copilot Studio?

 - Power Automate opens in a new window within the browser you're working on and opens with the templated input and output actions already built in Power Automate. At the same time, a popup opens in Microsoft Copilot Studio saying that the new window is open to configure Power Automate and to select Done to refresh Copilot Studio's Authoring Canvas

 - Microsoft Copilot Studio closes, and Power Automate Opens

 - A popup asks you to open Power Automate and configure your flow

3. You created a **Power Automate flow** that connects to **Microsoft Copilot Studio**, and has input and output variables. After selecting the flow in the Authoring Canvas in Microsoft Copilot Studio, what do you need to do?

 - Select which flow to use
 - Author custom YAML to connect to Power Automate
 - Select the input variable in Copilot Studio, which stores the data you send as input to the Power Automate flow, and then select the output variable in Copilot Studio that stores the data Power Automate sends back to Microsoft Copilot Studio.
