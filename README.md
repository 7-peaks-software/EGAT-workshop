# Microsoft Copilot Studio Workshop

This repository contains comprehensive workshop materials for learning how to create and manage intelligent chatbots using Microsoft Copilot Studio. The workshop is structured into multiple modules, each focusing on specific aspects of chatbot development.

## Workshop Structure

### 1. Planning Your First Chatbot
- Introduction to chatbot planning
- Understanding scenarios and prerequisites
- Best practices for chatbot design

### 2. Creating Your First Agent
- Testing the agent interface
- Creating your first topic
- Publishing to demo site
- Knowledge assessment
- Workshop summary

### 3. Unified Authoring Experience
- Working with entities and custom entities
- Managing variables and conditions
- Understanding topic nodes
- Configuring question behavior
- Implementing rich text responses
- Creating message variations
- Optimizing productivity
- Speech authoring capabilities
- Using the code view effectively

### 4. Building Flows with Power Automate
- Introduction to Power Automate integration
- Understanding Power Automate capabilities
- Hands-on flow building exercises

## Key Features Covered
- Microsoft Copilot Studio overview and interface
- Environment management
- Topic creation and management
- System fallback handling
- Testing and publishing workflows
- Natural Language Understanding (NLU)
- Generative answers implementation
- Document integration
- Custom data source configuration

## Prerequisites
- Access to Microsoft Copilot Studio
- Basic understanding of chatbot concepts
- Power Platform account access

## Getting Started
1. Start with the introduction module in `01-plan-first-chatbot-online-workshop`
2. Follow the sequential modules
3. Complete exercises in each section
4. Check your knowledge with provided assessments
5. Review summaries for key takeaways
