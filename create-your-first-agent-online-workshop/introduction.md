# Create Your First Agent Online Workshop

Learn how to create your first agent in Microsoft Copilot Studio.

## Learning objectives

In this module, you'll:

- Create a new agent by using the unified authoring canvas in Microsoft Copilot Studio.
- Create your first topic and add content to the topic, including rich text responses and more.
- Use the Copilot feature in Microsoft Copilot Studio to create a topic with AI.
- Learn how to test your agent in Microsoft Copilot Studio.
- Publish your agent to a demo website.
