# 5. Summary

By completing the Create your first agent in Microsoft Copilot Studio - Online workshop module, you've successfully learned how to:

- Access your Microsoft Copilot Studio preview agent that uses the latest unified authoring canvas.

- Review the new user interface (UI) and become familiar with how to navigate the agent maker experience.

- Create your first custom topic.

- Add nodes to your first custom topic.

- Use Copilot in Microsoft Copilot Studio to create a topic with AI.

- Test your agent within the Microsoft Copilot Studio testing tool.

- Publish and deploy your agent to the demo website.
